# 业务系统模版

```
本项目为精简版的vue admin，支持多项目构建，异步路由等功能；
vue admin地址：https://github.com/PanJiaChen/vue-element-admin
```

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn run dev
```

### Compiles and minifies for production
```
yarn run build
```

### Run your tests
```
yarn run test
```

### Lints and fixes files
```
yarn run lint
```

### Run your unit tests
```
yarn run test:unit
```
